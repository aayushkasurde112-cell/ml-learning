# Matplotlib

## Key concepts

- pyplot vs object-oriented API
- subplots, legends, labels

## Common operations

```python
import matplotlib.pyplot as plt

plt.figure(figsize=(8, 4))
plt.plot(x, y)
plt.xlabel("X")
plt.ylabel("Y")
plt.title("Title")
plt.show()
```
